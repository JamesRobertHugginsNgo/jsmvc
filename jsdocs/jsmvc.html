<static> eventfulPropertyDescriptors

An object property descriptor used to add eventful properties to an object.
Properties:
Name Type Description
definedBy_eventfulPropertyDescriptors object Adds a flag used to determin if the object was defined with jsmvc.eventfulPropertyDescriptors
eventfulEnabled object Adds a property representing the object's enable state
enableEventful object Adds a method used to set the enable state to enabled
disableEventful object Adds a method used to set the enable state to disabled
eventfulData object Adds a property to hold event related data
eventfulReferences object Adds a property to hold event related data of another object
addEvent object Adds a method for adding event handler to an event
removeEvents object Adds a method for removing event handler to an event
triggerEvents object Adds a method for triggering event handler of an event
addEmitterEvent object Adds a method for adding event handler to an event of another object
removeEmitterEvents object Adds a method for removing event handler to an event of another object
on object Alias to addEvent method
off object Alias to removeEvents method
trigger object Alias to triggerEvents method
listenTo object Alias to addEmitterEvent method
stopListening object Alias to removeEmitterEvents method
Example
const obj = {};
Object.defineProperties(obj, jsmvc.eventfulPropertyDescriptors);

// Add an event handler
obj.on('test', () => { console.log('test1'); });

// Add another event handler
const handler = () => { console.log('test2'); };
obj.on('test', handler);

// Remove event handler
obj.off('test', handler);

obj.trigger('test')
